# Руководство по SMS провайдерам

## 1. Mock Provider (для разработки) - ✅ УЖЕ ПОДКЛЮЧЕН

По умолчанию используется **Mock провайдер**, который логирует коды восстановления в консоль и логи.

### Как это работает:
- Код восстановления выводится в консоль с зеленым цветом
- Можно тестировать без настройки SMS сервиса
- Идеально для разработки и тестирования

### Использование:
```
SmsSettings:Provider = "Mock"
```

**Пример вывода в консоль:**
```
[SMS] Номер: +79991234567
[SMS] Текст: Ваш код для восстановления пароля: 123456. Код действителен 10 минут.
```

## 2. Twilio - Бесплатный Trial

Twilio предоставляет бесплатный trial с $5.50 кредитом для тестирования SMS.

### Шаги подключения:

#### 1. Регистрация
- Перейти на https://www.twilio.com/try-twilio
- Создать аккаунт (требуется номер телефона для верификации)
- Получить бесплатный trial номер (например, +1234567890)

#### 2. Получить учетные данные
- В консоли Twilio перейти в Account → API keys & tokens
- Скопировать:
  - **Account SID** (начинается с AC...)
  - **Auth Token** (длинная строка)
